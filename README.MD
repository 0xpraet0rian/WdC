# Walk Distance Calculator (WDC)

A QGIS plugin for estimating walking time and distance along user-defined paths, with customizable parameters such as age, physical condition, and terrain.

---

## Overview

The **Walk Distance Calculator (WDC)** is designed to simulate realistic human movement across geographic space. It allows users to draw paths directly on the map and calculate estimated walking time based on configurable parameters.

This tool is especially useful for:

- Archaeological and historical mobility analysis  
- Military logistics simulations (e.g., army movement)  
- Fieldwork planning and accessibility studies  
- General GIS-based route estimation  

The plugin focuses on **human-centered movement modeling**, going beyond simple distance calculations.

---

## Features

- 📏 **Interactive path drawing**  
  Draw custom routes directly in QGIS.

- ⏱️ **Walking time estimation**  
  Calculates travel time based on distance and speed.

- ⚙️ **Custom walking speeds**  
  Supports predefined profiles and user-defined values.

- 👥 **User profiles**  
  Adjust calculations based on:
  - Age group  
  - Height  
  - Training level (civilian, trained, soldier)

- 🏔️ **Terrain awareness (planned/partial)**  
  Designed to account for slope and terrain difficulty.

- 📤 **Export results (planned/partial)**  
  Output calculated data for further analysis.
